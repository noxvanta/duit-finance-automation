# 💰 DUIT (Data Uang In dan Out Terpadu)

**DUIT** adalah asisten keuangan berbasis AI yang mengotomatisasi pencatatan pengeluaran dan pemasukan langsung dari Telegram ke Google Sheets.

## 🚀 Fitur Utama
- **Automasi Total:** Input data lewat Telegram, otomatis tercatat di Google Sheets.
- **AI-Powered:** Menggunakan LLM (Gemini/Groq) untuk kategorisasi transaksi secara cerdas.
- **Real-time Logging:** Tidak perlu buka spreadsheet manual setiap kali jajan.

## 🛠️ Tech Stack
- **n8n:** Workflow automation engine.
- **Telegram Bot API:** Interface pengguna.
- **Google Sheets API:** Database penyimpanan data.
- **AI Integration:** Gemini dan Groq.
